# Paradise Nursery

**Project Name:** Paradise Nursery Shopping Application

**Description:**
Paradise Nursery is a front-end web application for browsing and purchasing houseplants. 
Users can view plants, add them to a shopping cart, adjust quantities, and see the total cost. 
The app includes a landing page, product listing page, and shopping cart page with dynamic functionality.

**Features:**
- Landing page with company info and background image
- Product listing page with plant images, names, prices, and add-to-cart buttons
- Shopping cart page with quantity adjustment, delete option, and total price calculation
- Header with dynamic cart count

**Technologies Used:**
- HTML, CSS, JavaScript
- React.js (for component-based architecture)
- GitHub Pages for deployment
